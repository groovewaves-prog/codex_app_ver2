🛡 Secure Review

The primary UI is now the Streamlit application. Run it from the repository root:

Start the UI

pip install -r requirements.txt
streamlit run streamlit_app.py

Or call the HTTP API directly

  • POST /api/preview — extract, sanitize, and run the local sensitivity gate. Returns per-document decisions.
  • POST /api/review — run the configured LLM provider. Any document with decision mask_and_continue must be confirmed via confirmMaskAndContinue or documentConfirmations in the request body.

Only sanitized text reaches external providers. Original content and replacement maps stay in server memory for the active session only.
